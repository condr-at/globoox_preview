# Reader Layout Docs

This folder contains the current source of truth for reader layout, pagination, typography, and diagnostics.

## Documents
- `architecture.md` — high-level rendering/pagination architecture.
- `pagination-rules.md` — pagination and keep-with-next rules.
- `rules-register.md` — centralized rules matrix with `as-is / to-be` status.
- `typography-and-hyphenation.md` — heading scale and hyphenation policy.
- `caching-and-versioning.md` — layout cache behavior and invalidation strategy.
- `debugging-playbook.md` — practical debugging flow for layout/pagination issues.

## Scope
This documentation describes the current behavior in `ReaderView` + `paginatorUtils`, not historical plans.
