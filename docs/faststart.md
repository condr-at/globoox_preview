По апи:
Структура данных: Книга содержит главы, глава содержит блоки контента.

Book
 └── Chapter[]
      └── ContentBlock[]
           └── translations: { lang: text }

Сущности:
1. Book 
{
  id: "uuid",
  title: "Мастер и Маргарита",
  author: "Михаил Булгаков",
  original_language: "ru",
  cover_url: "https://storage.../cover.jpg",
  available_languages: ["ru", "en", "de", "es"],
  created_at: "2025-01-15T..."
}

2. Chapter

{
  id: "uuid",
  book_id: "uuid",
  index: 0,              // порядок главы в книге
  title: "Глава 1. Никогда не разговаривайте с неизвестными"
}

3. ContentBlock

{
  id: "uuid",
  chapter_id: "uuid",
  position: 0,           // порядок внутри главы
  type: "paragraph",     // paragraph | heading | quote | image
  
  original_text: "Однажды весною, в час небывало жаркого заката...",
  
  translations: {
    "en": "One spring day, at the hour of an unusually hot sunset...",
    "de": "Eines Frühlingstages, zur Stunde eines ungewöhnlich heißen Sonnenuntergangs...",
    "es": null  // ещё не переведено
  }
}

Как фронтенд получает данные
Список книг:
GET /books → Book[]

Оглавление книги:
GET /books/{id}/chapters → Chapter[]

Контент главы на нужном языке:
GET /chapters/{id}/content?lang=en → ContentBlock[]
Ответ:
[
  { id: "...", position: 0, type: "heading", text: "Chapter 1" },
  { id: "...", position: 1, type: "paragraph", text: "One spring day..." },
  { id: "...", position: 2, type: "paragraph", text: "..." },
  { id: "...", position: 3, type: "image", url: "https://..." }
]

Типы блоков:
heading
Заголовок/подзаголовок
<h2> или <h3>

paragraph
Обычный текст
<p>

quote
Цитата, эпиграф
<blockquote>

image
Картинка
<img src={url}>
